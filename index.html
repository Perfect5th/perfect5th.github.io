What I do

Web Apps

I have extensive experience building web applications using the Python framework Django. I also have some experience developing in Ruby on Rails and NodeJS. My current work is as a full-stack web developer with Biolux Research, Ltd. in Vancouver, BC. This includes developing and maintaining a web portal, REST API, and online store as part of a small team. I'm always excited to work with new technologies and start new projects.

Web Design

I enjoy working with the lastest JavaScript, HTML, and CSS web standards, and focus on clean, intuitive design. I am very comfortable working with designers and UX specialists on all manner of projects.

Data Analysis

With my background in biochemistry, I started my career in a research-oriented role. I have a high-level working knowledge of statistics and data analysis, including visualization and working with large datasets. I currently work with Tableau, Jupyter, and R, and have familiarity with SQL.
