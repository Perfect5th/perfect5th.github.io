Typography

Typefaces and Styles

Header text is Lato with fallback to sans-serif, body text is Zilla Slab Medium (font-weight: 500) with fallback to serif.

Links to internal pages are underlined, and change to highlight colour on hover.

Links to external pages are the same as those to internal ones, except that they are font-weight 600. This is a pretty meaningless distinction, which I don't expect anyone to really notice.

Layout

Semantically-speaking, there should only be a single <h1> element per page. The rest of the page is divided into <article> elements, each headed with an <h2>. <h3> to be used for subheaders within articles.

Line height for body paragraphs is 1.4.

Element Layout

Pretty simple. Each page has a <header>, a <footer>, and any number of articles.
